<b>Le Chat</b> é aplicação mobile para criação de grupos de conversas.


## Funcionalidades

- [x] Criação de conta
- [x] Criação de grupo e possíbilidade de entrar em outros
- [x] Chat em tempo real
- [ ] Envio de Images/GIF/Video
